## 역할
너는 주로 리액트를 통해 웹을 만드는 시니어 프론트엔드 엔지니어야.

## 목적
유형 검사를 진행하고 답변에 따라 서로 다른 결과를 나타내주는 웹 페이지를 만들어야해.
질문은 총 12가지로 구성되어 있고 답변은 이지선다야. 검사 결과는 스마트 컨트랙트 개발자, 블록체인 리서처, Web3 PM, 커뮤니티 매니저 총 4가지로 분류돼.
백엔드를 사용하지 않아야하며 여러 사람이 한 PC로 검사를 순서대로 진행하기 때문에 “다음 사람 시작”에서 답변/세션을 무조건 초기화해야해.

## 예시
철수 검사 시작 > Q1. A 선택 / Q2. B 선택 / Q3. A 선택 / ... / => 검사 결과 : Web3 PM
철수 검사 데이터 초기화
영희 검사 시작 > Q1. B 선택 / Q2. A 선택 / Q3. B 선택 / ... / => 검사 결과 : 스마트 컨트랙트 개발자
영희 검사 데이터 초기화

## 핵심
1. React를 활용해서 개발
2. 백엔드 사용 X
3. "검사 끝" => “다음 사람 검사 시작” 에서 답변 데이터를 무조건 초기화
